# SKU Generator (Desktop App)

This is a simple Python desktop application that generates product SKUs using the format:

BRAND-NAME-PRODUCTTYPE-COLOR-SIZE

Users can manually enter the brand, product name, and colors, while product type and size are selected from dropdown menus. The app automatically removes spaces and special characters, converts everything to uppercase, and generates all SKU combinations when multiple colors or sizes are provided.

This project is still in progress and additional features and improvements will be added in the future.
